How to use a promo code

During registration, look for a field labelled "Promo Code", "Bonus Code" or "Referral Code". Enter the code exactly as shown above, including any capital letters. Some platforms are case-sensitive. If you miss the field during sign-up, check whether there is a bonus-code entry option in your account settings before making a first deposit.

If the code does not apply automatically after your deposit, contact the bookmaker's live chat support. Keep a screenshot of the code you used as evidence.

What happens if I don't use a code?

If a bookmaker requires a code and you do not enter it, the bonus will not activate. Some bookmakers will add the code after the fact if contacted promptly. Others will not. Always check the bonus terms at registration time to know whether a code is required.
